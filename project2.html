Campus Event Scheduler

Java Android Studio Google Maps API MySQL Material Design

Collaborated on a comprehensive campus event application built in Java using Android Studio. The app streamlines event discovery and management for students by integrating Google Maps API for precise event locations, a MySQL backend for robust event data management, and a modern user interface designed with Material Design principles.

Key Features

  • Interactive Map Integration: Implemented Google Maps API to display event locations with custom markers, allowing users to visualize campus events geographically and get directions.
  • Event Management System: Built a comprehensive backend using MySQL to handle event creation, updates, and queries, ensuring reliable data persistence and retrieval.
  • Material Design UI: Designed an intuitive and modern user interface following Material Design guidelines, enhancing user experience with smooth animations and responsive layouts.
  • Event Discovery: Developed browsing and search functionality to help students easily find events by date, category, location, or keywords.
  • User Authentication: Implemented secure user login and registration to enable personalized event tracking and preferences.

This collaborative project enhanced campus engagement by providing students with a centralized platform to discover and attend events.

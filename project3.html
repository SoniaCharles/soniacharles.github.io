Backend Developer | Capstone Project

Led backend development for a full-stack e-commerce simulation platform in collaboration with Meijer, working within a team of 5 developers under an Agile framework. Key Contributions

  • RESTful API Development: Led backend development in C#/.NET, creating and maintaining robust RESTful APIs to support cart operations, checkout processes, and mPerks rewards gamification.
  • Database Architecture: Designed and optimized SQL Server database schemas, stored procedures, and queries to ensure high-performance data retrieval and scalability.
  • Rewards & Gamification Logic: Integrated donation logic and rewards algorithms that increased simulated customer interaction by 20%, enhancing user engagement.
  • Testing & Quality Assurance: Conducted extensive unit testing and debugging using Swagger and Postman, improving backend reliability and reducing API error rates.
  • Frontend Collaboration: Worked closely with ReactJS frontend teammates to ensure seamless API consumption and effective data visualization.
  • Cloud Deployment: Deployed and monitored backend services on Microsoft Azure App Service with GitLab CI/CD pipelines, maintaining high uptime and efficient delivery processes.

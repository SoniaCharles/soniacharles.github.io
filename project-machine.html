Canadian Experience & Harold’s Incredible Machine

C++ Object-Oriented Programming Interactive Systems

This project focused on building an interactive system in C++ that allowed users to create, control, and operate machine-based components over adjustable time frames. Using object-oriented programming, I implemented features involving actors, positioning, timing, and machine interaction. The project strengthened my skills in software structure, feature implementation, and designing interactive user-driven behavior.

Key Contributions

  • Interactive Features: Implemented actor-based functionality that allowed users to set exact positions and angles for different components.
  • Time Frame Control: Added support for creating, adjusting, and deleting time frames to manage machine behavior over time.
  • Machine Construction: Helped develop a system that enabled users to build and operate custom machines through an interactive interface.
  • Integration: Integrated Harold’s machines into the project to expand functionality and make the system more dynamic and realistic.

Final Output

The final result was an interactive machine-based program where users could build, manipulate, and control components over customizable time frames. This project gave me valuable experience in implementing dynamic functionality in C++ and reinforced my understanding of object-oriented system design.

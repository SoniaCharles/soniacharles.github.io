Cloud-Based Customer Data Pipeline

Python SQL AWS S3 PySpark Pandas Git

This project delivered an end-to-end ETL pipeline for customer transaction data, built on Python and AWS. The pipeline ingests raw transaction records, cleans and standardizes them, and loads them into dimensional models designed for analytics. I used PySpark for scalable transformation and optimized SQL queries so the resulting data could support reporting and downstream machine-learning workflows.

Yelp Review Sentiment Classification

Python Machine Learning NLP Scikit-learn

This project built a multi-class sentiment classification system over a corpus of 10,000 Yelp reviews. I compared several classical machine-learning models and an NLP feature pipeline to see how preprocessing and model choice affect accuracy on noisy, real-world review text — turning raw reviews into clean, weighted features and benchmarking models against one another under cross-validation.

Key Contributions

  • Multi-Class Classification Pipelines: Built sentiment classification pipelines on 10,000 Yelp reviews using SVM, Naive Bayes, Gaussian MAP, and Random Forest models.
  • NLP Feature Engineering: Implemented TF-IDF vectorization to convert raw review text into weighted numerical features suitable for classical ML models.
  • Feature Selection & Dimensionality Reduction: Applied feature selection and dimensionality reduction to cut noise and keep the most informative signals from the text.
  • Model Benchmarking: Evaluated models under cross-validation, achieving 58.39% accuracy with SVM as the strongest performer across the compared approaches.

Final Outcome

The project produced a working sentiment classifier and a clear comparison of how classical models handle real review text, with SVM reaching 58.39% cross-validation accuracy. It strengthened my hands-on experience with the full machine-learning workflow — text preprocessing, feature engineering, model selection, and rigorous evaluation.

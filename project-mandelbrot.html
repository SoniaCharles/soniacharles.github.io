Massively Parallel Mandelbrot Renderer & GPU Microbenchmarks

C++ CUDA PTX Pthreads SIMD Parallel Computing

This project focused on implementing a high-performance Mandelbrot set renderer using multiple layers of parallel computing techniques. The implementation explored CPU vectorization, multi-core threading, instruction-level parallelism, and GPU acceleration with CUDA.

In addition to rendering the fractal, I developed GPU microbenchmarks to analyze instruction throughput and memory latency across different levels of GPU memory. The goal was to better understand how warp scheduling, memory access patterns, and instruction latency affect performance in modern GPUs.

Key Contributions

  • Parallel Mandelbrot Rendering: Implemented a Mandelbrot fractal renderer using multiple parallelization techniques including SIMD vectorization, instruction-level parallelism, and multi-threading with Pthreads.
  • GPU Acceleration: Ported the workload to CUDA to leverage warp-level parallelism across streaming multiprocessors, significantly improving computation speed compared to CPU implementations.
  • Microbenchmark Development: Designed CUDA and PTX microbenchmarks to measure floating-point throughput (FFMA) and memory latency across L1, L2, and global GPU memory.
  • Memory Access Analysis: Evaluated the performance impact of coalesced versus non-coalesced memory accesses and how memory hierarchy affects GPU throughput.

Final Outcome

The final implementation demonstrated how different layers of parallelism—from SIMD and threading on the CPU to warp-level execution on the GPU—can dramatically improve computational performance. Through benchmarking and experimentation, the project provided hands-on insight into GPU execution models, instruction latency, and memory behavior in high-performance computing systems.

Sudoku Game Implementation

C++ Object-Oriented Programming Game Design

This project involved developing a Sudoku game in C++ using object-oriented programming principles. Working alongside fellow engineers, I helped build an interactive game that included multiple difficulty levels, user-friendly gameplay features, and an automatic solving option. The project strengthened my experience in designing structured software, handling user interaction, and solving logic-based programming problems.

Key Contributions

  • Game Development: Helped develop a fully interactive Sudoku game with logic-driven gameplay and responsive user interaction.
  • Difficulty Levels: Implemented three difficulty settings to provide different gameplay experiences for beginner, intermediate, and advanced users.
  • Automatic Solver: Added an automatic solving feature to demonstrate algorithmic problem-solving and puzzle validation.
  • Team Collaboration: Worked collaboratively with fellow engineers to design features, test functionality, and improve the overall user experience.
